JavaScript is an easy to learn, lightwight interpreted programming language. It comes with most web browsers, even on mobile devices. Because all modern browsers support JavaScript, it can be inserted into web pages. Once inserted, the code allows the developer to manipulate various sorts of data - whether written into the page, pulled from the internet or sent from user input - and write that manipulated date into the web page as dynamically updated information.

This class is about exploring what can be done JavaScript, HTML, and Cascading Style Sheets are combined. Come with me and explore what can be done when these technologies are combined! We will focus, as much as possible within the parameters of the class, on using JavaScript to where appropriate, with the idea that sometimes other technologies are more suited to a gioven task. In other words, just cuz ya can do it with JavaScript that don't mean ya should do it with JavaScript!
